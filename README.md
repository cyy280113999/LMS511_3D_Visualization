# LMS511_3D_Visualization
Visualization of point clouds from a rotating LMS511 Lidar
激光雷达数据读取与三维显示

# Running:

需要设备:LMS511， 转台（淘宝）

设备如何摆放： lidar.pdf

旋转周期：20s，可修改

旋转方向：逆时针

# IDE: 
运行使用PyCharm

# Environment:
配置：安装miniconda，安装以下包

Conda: python3.6

pip : pyqt5

conda vtk=8.1
